# Aula12ED_5
## Atividade 5

5. Considere um Objeto Pessoa com Nome e idade. Criar uma função hash que coloque todos as pessoas inseridas em uma tabela de espalhamento de 3 posições [0 a 18 anos ; 19 a 45 anos ; 45 anos ou +]. Fazer um projeto Java que permita, na classe de controle, inicializar a tabela de espalhamento, inserir pessoas, excluir pessoas e buscar pessoas por seu nome. A função hash deve estar definida no objeto.
